## Using Help

Wwise Help includes reference topics that provide detailed information on each interface element in Wwise.

To open the reference topic pertaining to your current view or dialog box from within Wwise, do one of the following:

-   Click the Help icon in the title bar of any of the views or dialog boxes.
-   From the menu bar, click **Help** > **Wwise Help**.
-   Press **F1**.

## Getting Support

If you are experiencing difficulties, or think you may have detected a bug, ensure you are using the latest minor version of Wwise before proceeding. As bugs are resolved with each minor update, it's possible the issue has already been resolved.

If the issue persists, the following options are available:

# Using Help

-   Ask the community through the [Q&A section](https://www.audiokinetic.com/qa/) of our website.
-   Report the issue through the Audiokinetic Launcher by selecting **Help** > **Report a Bug**. Refer to [Reporting Wwise Bugs](https://www.audiokinetic.com/library/wwise_launcher/?source=InstallGuide&id=reporting_bugs) for details on the process.
-   Submit a support ticket using the Audiokinetic [Customer Portal](https://www.audiokinetic.com/customers/).
