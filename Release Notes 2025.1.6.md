## New Features

-   **WG-66056** It's now possible to use a channel configuration of 4.0.4 with Audio Busses, Soundseed Grain, and Reflect.
-   **WG-80877** New option added in Property Editor to show the Make-Up Gain as a fader instead of a slider.
-   **WG-81545** Motion plug-in now includes the Haptic Clip Player source plug-in, which supports `.haptic` files across haptic-enabled devices.
-   **WG-81547** Motion plug-in now supports Meta Quest Touch controllers on Android and Windows through the OpenXR API.
-   **WG-82670** Added the ability to interact with Voices and Busses in the CPU Timeline of the Advanced Profiler. When the CPU Timeline is able to resolve a Timer Scope to a Voice or Bus in the Wwise Project, you can double-click the Timer Scope to view details of the Voice or Bus and right-click the Timer Scope to open a relevant shortcut menu.

## Behavior Changes

-   **WG-19992** Volume fades no longer contribute to a playing sound's virtual status. This includes fades generated by actions (play, stop, pause, resume), and fades initiated by a container, for example, a Random Container with a crossfade.
-   **WG-82049** The None State now consistently appears as the first State of a State Group in several views.
-   **WG-82329** The monitoring curve of the Parametric EQ Effect no longer appears when there are no active instances of the Effect when profiling a game or when profiling is inactive.

## Miscellaneous Changes

-   **WG-80684** Significantly improved drawing performance of graph-based editors in Wwise Authoring. This also resolves some visual artifacts with marquee selection in the RTPC graphs, AK Convolution plug-in, and Mastering Suite plug-in.
-   **WG-82083** Updated Android NDK to r27d.

## Bug Fixes

-   **WG-75245** Fixed: Excluding a sound object during playback doesn't stop its source plug-in.
-   **WG-81407** Fixed: (Acoustics) In some cases, paths between a Room and the listener are calculated but not used.
-   **WG-81593** Fixed: (Acoustics) Direct paths can escape through gaps in Portals without applying Transmission Loss.
-   **WG-81739** Fixed: Clicking the Help button in the Property Editor Favorites dialog opens the incorrect page in Wwise Help.
-   **WG-81833** Fixed: Crash in the Game Object 3D Viewer when profiling a game that has an unnamed spatial audio Portal.
-   **WG-82053** Fixed: Crash when selecting **Open in New Tab** in the shortcut menu of a Music Switch Container.
-   **WG-82078** Fixed: Crash when Replay encounters a `PostEvent` command with a callback.
-   **WG-82263** Fixed: Not possible to scroll down in the CPU Timeline of the Advanced Profiler when viewing profiling data of a runtime with many threads, but no timing data for each thread.
-   **WG-82290** Fixed: Tone Generator plug-in's random number generator behaves non-deterministically when running in multi-core mode.
-   **WG-82383** Fixed: (Mac Authoring) Crash when moving a band in the Parametric EQ plug-in.
-   **WG-82484** Fixed: Function `SetRTPCValueByPlayingID` fails when called during the audio frame in which the playing ID was created.
-   **WG-82486** Fixed: 3D panning of positioned sources or busses with some height channel data, such as 4.0.4 or 5.1.4, does not include some of the input channels during mixing. Note that one consequence of this change is that some existing content using 4.0.4 or 5.1.4 configs that had highly correlated data across channels, in such a way that the missing input was not directly noticed, might have their apparent loudness increased.
-   **WG-82534** Fixed: (Acoustics) Possible crash in `AkGeometrySet/AkRTree` when running in low-memory conditions.
-   **WG-82589** Fixed: Continuous Random and Sequence Containers don't always finish playback. This is most likely to occur when playing a child container that doesn't play any voices.
-   **WG-82633** Fixed: Unicode text in markers are sometimes treated as binary data.
-   **WG-82637** Fixed: Assert when a **Reset Bus Volume All** Action is triggered with game-object scope.

# New Features

-   **WG-68133** Fixed: (Mac Authoring) Lack of stability when changing an object name in the project hierarchy.
-   **WG-79546** Fixed: Crash when reading the cache data for an audio waveform.
-   **WG-81644** Fixed: (Acoustics) Calling `SetPortal` copies the Portal name string, even when unchanged, resulting in an unnecessary allocation.
-   **WG-81680** Fixed: AK ASIO Output plug-in is not allowed to retry device initialization after a recoverable initialization error.
-   **WG-81837** Fixed: Crash when reading a `.wem` file that has a header spanning multiple read buffers, and the I/O Hook doesn't read a stream with consistent buffer sizes.
-   **WG-81943** Fixed: Possible crash when loading a project that contains missing references.
-   **WG-82058** Fixed: Output Bus DSF filter value is not stored in SoundBanks.
-   **WG-82159** Fixed: Crash when copying Effects or States.
-   **WG-82267** Fixed: Different converted audio files are generated depending on when the conversion takes place. If a file is edited in the Source Editor and then converted before closing the project, the converted file might differ from the converted file created when the project is closed between editing the file and conversion.
-   **WG-82293** Fixed: `ResetRTPCValueWithTransition` API call displays wrong transition time in Capture Log.
-   **WG-82320** Fixed: Check boxes that enable the individual modules of the Mastering Suite don't accurately reflect the state of the properties.
-   **WG-82322** Fixed: Property values in the Parametric EQ module of the Mastering Suite plug-in don't update when moving bands in the graph.
-   **WG-82335** Fixed: (Acoustics) Feedback loop between Room aux busses.
-   **WG-82477** Fixed: SoundBank Generation log contains messages referring to missing Auto-Defined Soundbanks.
-   **WG-82479** Fixed: SoundBank Generation log reports media duplication for Auto-Defined Soundbanks.
-   **WG-82495** Fixed: Crash when using Blend Containers within Switch Containers and changing the Switch and RTPC simultaneously.
-   **WG-82528** Fixed: Crash when renaming Sidechain Mixes or Game Parameters referenced by plug-ins.
-   **WG-82597** Fixed: Random Containers and Sequence Containers can start too many playbacks.
-   **WG-82646** Fixed: Hang in `[AK::StreamMgr](https://www.audiokinetic.com/en/public-library/2025.1.9_9197/?source=SDK&id=namespace_a_k_1_1_stream_mgr.html "Audiokinetic Stream Manager's implementation-specific interfaces of the Low-Level IO submodule.")` when using offline rendering.
-   **WG-82680** Fixed: Erroneous "Media Not Found" error when generating Auto-Defined SoundBanks with Effect plug-ins on an audio bus.
-   **WG-82721** Fixed: Hang when using Random or Sequence Containers in Continuous mode with a sample-accurate Transition and voice limits.
-   **WG-82894** Fixed: Sidechain Effects and Sources send or receive Sidechain Mixes with an incorrect scope when using the **Profile and Edit** options to connect Wwise to a game.

## Documentation Improvements

-   **WG-82456** Corrected translation to Korean of the terms "propagate" and "propagation" when referring to sound.
